MERA Files API Reference

Functions for saving, loading, and converting MERA data files.

File Operations

  • savedata - Save data to MERA format
  • loaddata - Load MERA format data
  • viewdata - Inspect MERA file contents
  • infodata - Get MERA file information

Batch Operations

  • batch_convert_mera - Convert multiple files
  • interactive_mera_converter - Interactive conversion tool

File Optimization

  • optimize_mera_io - Optimize I/O performance
  • configure_mera_io - Configure I/O settings
  • show_mera_config - Display current configuration
  • reset_mera_io - Reset I/O settings
  • benchmark_mera_io - Benchmark I/O performance
  • mera_io_status - Check I/O status

Conversion Functions

  • Data format conversion
  • Export capabilities
  • Import from other formats

For complete function documentation, see the Complete API Reference.
